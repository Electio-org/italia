# Usage notes

Questa build adotta un approccio più vicino a GERDA: dataset-first, codebook esplicito, coverage e note d’uso sempre visibili.

- La base geometrica può essere scelta separatamente dall’anno osservato.
- La modalità armonizzata va usata solo se il bundle contiene righe harmonized vere.
- Le quote di partito possono restare parziali se i risultati long non sono esaustivi.
